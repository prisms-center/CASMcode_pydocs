is_equivalent_chain#

casm.tools.map.methods.is_equivalent_chain(
chain_A: list[Structure],
chain_B: list[Structure],
) → bool[source]#

Check if two chains of structures are equivalent.

Parameters:

  • chain_A (list[Structure]) – The first chain of structures. Expects all structures to be in canonical form.

  • chain_B (list[Structure]) – The second chain of structures. Expects all structures to be in canonical form.

Return type:

True if the chains are equivalent, False otherwise.
