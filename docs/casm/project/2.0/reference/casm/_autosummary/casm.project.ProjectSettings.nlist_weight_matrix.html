nlist_weight_matrix#

ProjectSettings.nlist_weight_matrix#

Used in determining the order of unit cells in the neighbor list.

The default value is determined from the prim lattice vectors. Typically does not need to be modified. If for some reason it is modified, all basis set source code must be regenerated.

Type:

numpy.ndarray[numpy.int[3,3]]
