multi_occ_delta_value#

ClusterExpansion.multi_occ_delta_value(
self: ClusterExpansion,
linear_site_index: list[int],
new_occ: list[int],
) → float#

Calculate the change in cluster expansion value due to a change in multiple occupant DoF values

Parameters:

  • key (str) – The type of local DoF that is changed.

  • linear_site_index (list[int]) – Linear site indices where occupant DoF values are changed.

  • new_occ (list[int]) – The value the occupant DoF are changed to.
