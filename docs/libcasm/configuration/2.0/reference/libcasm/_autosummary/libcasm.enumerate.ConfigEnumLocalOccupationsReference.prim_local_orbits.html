prim_local_orbits#

ConfigEnumLocalOccupationsReference.prim_local_orbits#

The local-cluster orbits about each equivalent event in the origin unit cell.

The local orbits for the equivalent events, where prim_local_orbits[i_event][i_local_orbit][i_cluster] is the i_cluster-th cluster in the i_local_orbit-th local-cluster orbit about event_info.event_prim_info.events[i_event].

Type:

list[list[list[Cluster]]]
